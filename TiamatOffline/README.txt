В папката TiamatOffline са оставени файловете, чрез които е създаден моделът, който се използва в онлайн
търгуването. При желание за проверка и debug трябва да се изтеглят следните библиотеки:

1) Трябва да имате инсталиран python3.12 - https://www.python.org/downloads/release/python-3129/

2) Всички използвани python библиотеки, описани в requirements.txt, трябва да се изтеглят чрез Command Prompt.

- cd (TiamatOffline)
- pip install -r requirements.txt

3) Да се изтегли QuestDB

- https://questdb.com/download/

4) Изтегля се ZIP файл, който трябва да се разархивира

5) Трябва да се стартира базата данни QuestDB:

- Отворете разархивираната папка
- Отворете bin
- Отворете questdb.exe

6) Вкарване на нужните данни:

- Използвайте Command Prompt и локализирайте до TiamatOffline (cd Path/TiamatOffline)
- Пуснете news_tz.py през командата (python3.12 news_tz.py) 
    - Задължително трябва в същата папка да съществува market_news.csv 
- Пуснете dataset_without_indicators.py през командата (python3.12 dataset_without_indicators.py) 
    - Задължително трябва в същата папка да съществува XAUUSD_M1_RAW.csv 

7) Създаване/получаване на най-добрият модел

- Пуснете hyperparameters.py през командата (python3.12 hyperparameters.py) 
    - Тази команда ще започне множество от "trial-и" за намиране на най-оптимален и добър изкуствен модел, който да може да търгува в реално време
    - Възможно е забиване и лаг, ако не желаете да намерите собствен модел, може да отидете при TiamatOnline, където има вече намерен такъв модел, заедно с графа, доказваща, че в действителност е добър.
